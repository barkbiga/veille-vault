# How Cyberattacks Can Turn Battery Farms Into Grid Blackouts

| | |
|---|---|
| **Pertinence** | 2/5 |
| **Catégorie** | Sécurité |
| **Source** | [BankInfoSecurity](https://www.bankinfosecurity.com/how-cyberattacks-turn-battery-farms-into-grid-blackouts-a-31191) |

## Résumé
Incident de décembre 2025 en Pologne : des attaquants ont perturbé l'équilibre du réseau électrique en ciblant des systèmes de stockage par batteries (BESS). Le cas illustre que la décentralisation de l'énergie (pour supporter les workloads IA) crée des surfaces d'attaque OT nouvelles, mal protégées. Connexion indirecte avec le secteur financier : la dépendance des data centers bancaires à ces infrastructures énergétiques décentralisées est un risque de continuité sous-estimé dans les analyses DORA.

🌱 SIGNAL FAIBLE — ↔️ Infrastructure critique → Services financiers : la résilience opérationnelle DORA (Art. 11) devrait intégrer les dépendances énergétiques OT comme vecteur de risque systémique.

## Actions recommandées
- [ ] Vérifier si les BIA (Business Impact Analysis) des clients bancaires couvrent la dépendance aux BESS/smart grid dans leur analyse de risque tiers DORA
