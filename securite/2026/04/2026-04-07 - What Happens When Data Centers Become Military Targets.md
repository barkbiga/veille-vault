# What Happens When Data Centers Become Military Targets?

| | |
|---|---|
| **Pertinence** | 2.5/5 |
| **Catégorie** | Sécurité |

## Résumé
Le ciblage de datacenters cloud commerciaux au Moyen-Orient introduit une nouvelle variable dans les plans de continuité : le risque militaire/géopolitique direct sur l'infrastructure physique cloud. Pour les services financiers européens sous DORA, cela interroge la dépendance aux hyperscalers et la notion de "exit strategy" réglementaire. L'article reste relativement superficiel (appel à revoir les BCP sans framework précis) mais le signal est pertinent dans un contexte DORA ICT risk management. 🌱 SIGNAL FAIBLE — la géopolitisation du cloud infrastructure n'est pas encore intégrée dans les frameworks de gestion des risques ICT bancaires européens.

↔️ Paiement → Assurance — La concentration cloud des assureurs IARD (catastrophe modeling, pricing) est aussi exposée ; sujet EIOPA encore émergent.

## Actions recommandées
- [ ] Vérifier si les EIOPA Guidelines on ICT Risk adressent le risque de ciblage militaire des infras cloud
- [ ] Note mentale : angle DORA article 28 (concentration risk) appliqué au scénario géopolitique
