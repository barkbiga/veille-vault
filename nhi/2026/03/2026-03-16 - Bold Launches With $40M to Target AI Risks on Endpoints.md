# Bold Launches With $40M to Target AI Risks on Endpoints

| | |
|---|---|
| **Pertinence** | 2.5/5 |
| **Catégorie** | NHI |

## Résumé
⚠️ Vendor (annonce de lancement startup) — Bold Security lève 40M$ pour sécuriser les endpoints à l'ère des agents IA, argumentant que les modèles cloud-heavy ne scalent pas pour les grandes entreprises. L'insight structurel est réel : les agents IA accèdent à des données localement (fichiers, apps, activité device), créant des angles morts que les contrôles cloud-centriques ne couvrent pas. 🌱 SIGNAL FAIBLE — soulève la question de l'identité des agents IA au niveau endpoint, un angle peu traité vs. l'identité des agents au niveau API/réseau. Contra-argument : le modèle endpoint-first pour les agents IA peut recréer les silos de visibilité que Zero Trust cherche précisément à éliminer — le risque est de revenir à une sécurité périmétrique déguisée.

## Actions recommandées
- [ ] Surveiller l'approche technique de Bold (comment gèrent-ils l'identité des agents locaux vs. les workload identities cloud ?)
- [ ] Creuser la tension cloud-centric vs. endpoint-centric pour la sécurité des agents IA dans le contexte du livre
