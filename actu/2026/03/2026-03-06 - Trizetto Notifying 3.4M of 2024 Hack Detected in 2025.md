# Trizetto Notifying 3.4M of 2024 Hack Detected in 2025

| | |
|---|---|

| **Catégorie** | Actu |

## Résumé
Trizetto Provider Solutions notifie 3,4 millions de personnes d'une compromission détectée en octobre 2025 mais initiée près d'un an plus tôt. L'incident concerne des données d'assurance santé de clients. Le délai de détection (~12 mois) illustre la persistance des acteurs malveillants dans les environnements de revenue cycle management. Pertinent comme signal de risque tiers dans les chaînes de traitement de données de santé, mais pas d'angle API/IAM direct.

## Actions recommandées
- [ ] Utiliser comme cas d'école pour argumenter la nécessité de détection comportementale sur les API B2B (long dwell time)
- [ ] Revoir les SLAs de détection dans les contrats tiers (secteur santé/assurance)
