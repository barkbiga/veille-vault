## Résumé
🔄 CONTRARIAN — Le titre lui-même challenge l'axiome "L'identité est le nouveau périmètre de sécurité" : si le modèle d'identité est fondamentalement cassé, construire sa sécurité sur ce périmètre est une fondation fragile. Le webinar réunit des leaders fintech et gaming pour diagnostiquer les limites structurelles des modèles IAM/CIAM actuels. Contenu résumé non disponible (webinar), mais la thèse centrale interpelle directement les architectes qui misent tout sur l'identité comme ligne de défense ultime.

## Actions recommandées
- [ ] Visionner le webinar pour identifier les arguments concrets contre le modèle identité-centrique
- [ ] Évaluer dans quelle mesure les APIs bancaires exposées restent vulnérables même avec une identité forte (ex : token theft, BOLA, BFLA)
- [ ] Confronter la thèse avec les retours d'expérience open banking pour challenger l'axiome n°4
