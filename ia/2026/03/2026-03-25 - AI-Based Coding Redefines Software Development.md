# AI-Based Coding Redefines Software Development

| | |
|---|---|

## Résumé
Jeetu Patel (Cisco) décrit un monde où les agents de coding refactorisent du code legacy d'entreprise à une vitesse et une échelle inatteignables par des équipes humaines. La vision "everyone will be a manager of agents" est pertinente pour le contexte du livre en cours sur la sécurité des APIs à l'ère des agents IA : si les agents écrivent le code, qui gère leur identité, leurs credentials, leurs permissions pendant le cycle CI/CD ? Signal faible sur la surface d'attaque NHI dans les pipelines DevSecOps. Contenu toutefois très haut niveau, pas de substance architecturale.

## Actions recommandées
- [ ] Creuser : si des agents IA génèrent et déploient du code, comment gouverner leurs NHI dans les pipelines CI/CD ? Angle potentiel pour le livre.
