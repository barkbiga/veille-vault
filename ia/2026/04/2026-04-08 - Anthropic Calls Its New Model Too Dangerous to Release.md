# Anthropic Calls Its New Model Too Dangerous to Release

| | |
|---|---|
| **Pertinence** | 2.5/5 |
| **Catégorie** | IA |
| **Source** | [BankInfoSecurity](https://www.bankinfosecurity.com/anthropic-calls-its-new-model-too-dangerous-to-release-a-31361) |

## Résumé
Anthropique affirme avoir développé Claude Mythos Preview, un modèle IA capable de trouver des milliers de vulnérabilités haute-sévérité — jugé trop dangereux pour une diffusion publique. Signal faible pour le livre "Securing APIs in the Age of AI Agents" : un modèle IA autonome capable de découvrir des CVE à grande échelle change radicalement la surface d'attaque sur les APIs exposées. La rétention volontaire d'un modèle par son créateur est un signal de gouvernance inédit. Pas d'analyse architecturale profonde dans l'article tel que résumé.

🌱 SIGNAL FAIBLE : Le gap entre capacité offensive IA et défense IA sur les APIs financières va s'accélérer. Les NHI (agents, service accounts) deviennent vecteurs d'exploitation automatisée.

## Actions recommandées
- [ ] Intégrer dans le chapitre threat modeling du livre : modèles IA comme outil d'attaque automatisée sur APIs
- [ ] Surveiller les détails techniques de Mythos Preview si publication partielle
